### OVERVIEW
Azure Media Services는 비디오 스트리밍을 구현하고 접근성과 배포를 향상시키며, 
콘텐츠를 분석하는 등 여러 작업을 수행하는 솔루션을 빌드할 수 있게 하는 클라우드 기반 플랫폼이다.
이 플랫폼을 토대로 동영상 스트리밍을 웹 사이트 형태로 하여 보다 발전된 영상 플랫폼을 구현해보고자 한다.</br></br>
